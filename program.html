Programming Tips:

Throughout our collective experience, we have learned many useful tips and tricks to use while programming a robot. Here are a few of the most useful, in our opinion.

Perseverance is Key

In our experience, the best way to succeed in programming is to keep on trying. More often than not, the program fails and does not work. Don't worry, it is extremely common, and you often have to spend an hour to find a simplistic logic mistake. You will find that programming gets quite frustrating at times, but the end result is always worth it. As long as you keep trying and learning, you will be successful in programming your robot.

Learning the Basics

When we started learning to program, we started off not knowing much about the variants of C++ used to program in VEX Robotics. To gain a footing, we researched many premade programs by VEX and other VEX Robotics teams, as well as watched many tutorials on YouTube. Using the plentiful resources online is a great way to learn the outline of a program, the format of programming, and overall, how to program for yourself.

Use Your Own Code

In our opinion, it is bad practive to use premade programs for your own competition robot. While these programs can be beneficial when learning how to program, you have lost the experience of building your own code if you decide to use these resources for your own code. If an issue with the code or a bug springs up unexpectedly, you need to be able to quickly fix it, and not knowing your code would slow that down tremendously.
